Optical Head Tracking and Stereoscopic Rendering

A virtual reality project combining optical tracking, computer vision and stereoscopic rendering through OpenGL and ArUco markers.

Technologies C++, OpenGL, OpenCV, ArUco, GLM
Field Virtual Reality and Computer Vision
Role Development, calibration and testing

Project overview

The project explores the interaction between physical markers and a virtual 3D environment. A webcam detects ArUco markers and extracts their position and orientation in real time.

This information is used to control both a virtual camera and a 3D object inside an OpenGL scene. The project also implements anaglyph stereoscopic rendering using toe-in and off-axis projection methods.

Technical process

The webcam was first calibrated to obtain its intrinsic parameters. The ArUco library was then used to detect markers and calculate their translation and rotation vectors.

These transformations were converted from the OpenCV coordinate system to the OpenGL coordinate system and applied to the virtual camera and the 3D object.

Project demonstrations

Camera tracking

The physical marker controls the position and orientation of the virtual camera.

Object tracking

A second marker controls the translation and rotation of a 3D object.

Toe-in stereoscopy

Stereoscopic rendering using two cameras oriented toward the same convergence point.

Off-axis stereoscopy

Stereoscopic rendering using parallel cameras and asymmetric projection frustums.
